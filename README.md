# IMDb Movie Scraper

A small Python project that pulls IMDb **Top Rated Movies** data via the public GraphQL API, saves it to CSV, and analyzes it with **Pandas** and **Matplotlib**.

## How it works

- **Scraper** (`scraper/imdb_scraper.py`) sends a GraphQL query to `https://graphql.imdb.com/` for the `TOP_RATED_MOVIES` chart (first **50** titles). It does **not** parse HTML; data comes from JSON responses.
- **Analysis** (`analysis/analysis.py`) reads the scraper’s CSV, prints summary stats (null counts, top titles by rating, yearly average ratings), splits `genres` into lists for the cleaned export, saves a horizontal bar chart of the top 10 by rating, and writes **`data/movies_cleaned.csv`**.

## Folder structure

```
IMDb-Movie-Scraper/
├── scraper/
│   └── imdb_scraper.py      # Fetch chart → data/movies_upgraded1.csv
├── analysis/
│   ├── analysis.py          # EDA + charts + data/movies_cleaned.csv
│   └── top10_movies.png     # Generated when you run analysis
├── data/
│   ├── movies_upgraded1.csv # Main output from the scraper (create by running scraper)
│   ├── movies_cleaned.csv   # Generated by analysis (after genres split to lists)
│   └── movies.csv           # Legacy / sample (optional)
├── requirements.txt
└── README.md
```

## CSV columns (scraper output)

| Column    | Description                                      |
| --------- | ------------------------------------------------ |
| `title`   | Movie title                                      |
| `rating`  | IMDb aggregate rating                            |
| `year`    | Release year (may be empty if missing from API)  |
| `runtime` | Length in **seconds** (may be empty)             |
| `genres`  | Comma-separated genre labels                     |
| `image`   | Poster image URL                                 |

If `data/movies_upgraded1.csv` is open in Excel or locked (e.g. OneDrive), the scraper saves a **timestamped** file under `data/` instead and prints the path.

## Setup

From the project root:

```bash
pip install -r requirements.txt
```

Dependencies used by the current code: **requests**, **pandas**, **matplotlib**. `beautifulsoup4` is listed in `requirements.txt` but is **not** used by the GraphQL scraper.

## Run the scraper

From the project root:

```bash
python scraper/imdb_scraper.py
```

This writes **`data/movies_upgraded1.csv`** (or a fallback `data/movies_YYYYMMDD_HHMMSS.csv` if the main file is locked).

## Run the analysis

Requires **`data/movies_upgraded1.csv`** (run the scraper first, or place a compatible CSV in `data/`).

From the project root:

```bash
python analysis/analysis.py
```

This prints EDA output to the console, saves **`analysis/top10_movies.png`**, and writes **`data/movies_cleaned.csv`** (with derived `runtime_minutes` / `runtime_hours` and `genres` as list cells).
